====== git ======
Created 목요일 14 6월 2018

ls -al 
현재 폴더의 내용을 보여준다.

git init 
초기화한다.(마스터가 생긴다.)

git status
git의 상태를 보여준다.

git add --all .
스테이징한다.

git --cached 
언스테이징 한다 .

git config --global user.email "you@example.com"
git config --global user.name "Your Name"
사용자 이메일과 성명을 적는다.

git commit -m "zimnote 시작"
내용을 적고 커밋한다.

 git remote add origin https://github.com/dlalstn604/zimnote.git
 git push -u origin master
