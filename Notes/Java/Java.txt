컴파일
프로그래머가 작성한 소스코드를 컴퓨터가 이해할 수 있는 언어로 바꾸는 행위

실행과정 : 
1. .java 파일을 .class 파일로 바꾸어 주는 컴파일 단계
2. .class 파일을 실행시키는 실행단계

{{.\pasted_image.png}}

class 확장자? 로 바꾸는 이유:
java는 모든 운영체제에서 사용할수 있게 만들기 위해
JVM이라는 중간단계가 있으므로 속도가 느리지만 어떤 OS에서든지 사용할 수 있다.

src 폴더 =  .java 파일이 저장
bin 폴더 = .class파일이 저장

스크립트 언어 : 스크립트 번역기가 읽으면서 실행

컴파일 언어 : 바이너리 코드로 클래스 파일을 만든다 
java 파일은 읽을 수 있지만 class파일은 읽을 수 없다.
JRE : class 파일을 받아서 읽는다.

자바파일 : java = 자바소스 : class
